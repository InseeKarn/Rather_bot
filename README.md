# Rather_bot Dev by InseeKarn

Rather_bot is a Python-based automated video generator that creates **"Would You Rather?"** style videos. It automatically combines text-to-speech, random question selection, AI image search, text overlays, background music, and YouTube uploads.

---

## Features

- Randomly selects questions from a JSON database.
- Converts questions to speech using **Google Text-to-Speech (gTTS)**.
- Fetches images automatically using **Unsplash API**.
- Generates text overlays with question options and random percentages.
- Adds background music and sound effects.
- Compiles high-quality videos using **MoviePy**.
- Avoids repeating questions.
- Auto uploads **YouTube Shorts** via **YouTube API**.
- Sends Discord notifications via webhook.
- Fully automated with **GitHub Actions**.
